SESSION 32

ふつうけい

Plain (short) form

GRAMMAR ❶ — Plain form, present

ふつうけい（げんざい）Plain form — present tense

EXAMPLE — ていねい → ふつう
たべtabeますmasu
↓ drop ／ change
たべtabeるru　(dictionary form)

The plain present affirmative is just the dictionary form. For ru-verbs, drop ます and add る: たべます → たべる. The negative is the ない-form: たべない. The red part is what changes.

GRAMMAR ❶ — more examples

れい：ふつうけい（げんざい）Present: dictionary / ない-form

〜ます　→　じしょけい ／ ない-form
のみます　→　のむ　／　のまないnomimasu → nomu / nomanai
かきます　→　かく　／　かかないkakimasu → kaku / kakanai
はなします　→　はなす　／　はなさないhanashimasu → hanasu / hanasanai
みます　→　みる　／　みないmimasu → miru / minai
します　→　する　／　しないshimasu → suru / shinai
きます　→　くる　／　こないkimasu → kuru / konai

する and くる are irregular. Everything in red is the part that changes.

GRAMMAR ❷ — Plain form, past

ふつうけい（かこ）Plain form — past tense

EXAMPLE
たべtabeるru
↓
たべtabeたta　(＝ past)

Plain past = the た-form. It has the same shape as the て-form, but with た/だ: たべて → たべた. Negative past: なかった → たべなかった.

GRAMMAR ❷ — more examples

れい：ふつうけい（かこ）Past: た-form / なかった

じしょけい　→　かこ ＋ ／ かこ −
のむ　→　のんだ　／　のまなかったnomu → nonda / nomanakatta
かく　→　かいた　／　かかなかったkaku → kaita / kakanakatta
たべる　→　たべた　／　たべなかったtaberu → tabeta / tabenakatta
する　→　した　／　しなかったsuru → shita / shinakatta
くる　→　きた　／　こなかったkuru → kita / konakatta

The past affirmative is the た-form (like the て-form: のんで→のんだ, かいて→かいた).

GRAMMAR ❸ — Adjectives & nouns

けいようし・めいしPlain form: adjectives & nouns

い-adjective
たかtakaいi
たかtakaかったkatta past
な-adj ／ noun
しずかshizukaだda
しずかshizukaだったdatta past

い-adjectives keep their normal forms (たかい ／ たかかった). な-adjectives and nouns add だ in the plain present: しずかだ, がくせいだ — this だ is the big difference from です.

GRAMMAR ❸ — all four forms

れい：ぜんぶの かたちpresent +/− ・ past +/−

い-adj（たかい）
たかい ／ たかくない ／ たかかった ／ たかくなかったtakai / takakunai / takakatta / takakunakatta
な-adj ／ noun（しずか・がくせい）
しずかだ ／ しずかじゃない ／ しずかだった ／ しずかじゃなかったshizukada / shizukajanai / shizukadatta / shizukajanakatta
がくせいだ ／ がくせいじゃない ／ がくせいだったgakuseida / gakuseijanai / gakuseidatta

Watch out: い-adjectives never take だ (×たかいだ). Only な-adjectives & nouns take だ.

SESSION 33

〜とおもいます

“I think that ~”

GRAMMAR ❹ — 〜とおもいます

〜と おもいます“I think that ~”

EXAMPLE
あしたashita あめame がga ふるfuru とto おもいomoiますmasu。

Use [plain form] ＋ と おもいます to say “I think that ~.” The part before と must be in the plain form: ふる (not ふります). → “I think it will rain tomorrow.”

この えいが・おもしろい？おもしろい と おもいます

GRAMMAR ❺ — 〜といっていました

〜と いっていました“(Someone) said that ~”

EXAMPLE
トムTomu さんsan はwa あしたashita くるkuru とto いってitteいましたimashita。

[plain form] ＋ と いっていました = “(someone) said that ~.” Report another person’s words. → “Tom said that he will come tomorrow.”

SESSION 34

〜のが すき

“like / be good at doing ~”

GRAMMAR ❻ — 〜のが すき

どうし ＋ のが すきVerb + のが すき / じょうず / へた

EXAMPLE
りょうりryouri をo するsuru のnoがga すきsukiですdesu。

Add の to a dictionary-form verb to make it a noun, then use が すき / じょうず / へた: するのが すきです = “I like cooking.” Also じょうず (good at) / へた (bad at).

GRAMMAR ❼ — なにか / なにも

なにか ／ なにも“something” / “nothing”

なにかnanika たべtabeましたmashitaか。
↓
いいえiie、なにもnanimo たべtabeませんでしたmasendeshita。

なにか = “something / anything” (in questions). なにも ＋ negative = “nothing at all.” Drop を after なにか / なにも.

SESSION 35

カジュアル・〜ないでください

Casual speech & “please don’t ~”

GRAMMAR ❽ — 〜ないでください

〜ないで ください“Please don’t ~”

EXAMPLE
ここkoko でde しゃしんshashin をo とらtoraないnaiでde くださいkudasai。

ない-form ＋ で ください = “please don’t ~.” とる → とらない → とらないでください. → “Please don’t take photos here.”

GRAMMAR ❾ — Casual speech

ためぐちCasual speech (drop です／ます)

ていねい → カジュアル
げんきgenkiですかdesu ka？　→　げんきgenki？
── うんun、げんきgenki。 ／ ううんuun、げんきgenkiじゃないjanai。

Among friends, drop です／ます and use the plain form. Rising intonation replaces か. うん = casual “yes,” ううん = casual “no.”
